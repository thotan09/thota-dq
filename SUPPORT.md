## Getting help

| Channel | Use for |
|---|---|
| [GitHub Issues](https://github.com/thotan09/thota-dq/issues) | Bug reports, feature requests |
| [GitHub Discussions](https://github.com/thotan09/thota-dq/discussions) | Questions, ideas, show-and-tell |
| [Stack Overflow `thota-dq`](https://stackoverflow.com/questions/tagged/thota-dq) | How-to questions |

## Before opening an issue

1. Search [existing issues](https://github.com/thotan09/thota-dq/issues) — your question may already be answered
2. Check the [documentation](https://thotan09.github.io/thota-dq)
3. Include your `thota-dq` version (`pip show thota-dq`), Python version, and a minimal reproducible example

## GitHub Action support

For issues specific to the `thotan09/thota-dq` GitHub Action, include:

- The full workflow YAML (redact secrets)
- The step output / error message
- The `thota-dq` version pinned in your workflow

# Support

## Documentation

Full documentation is at **[aegis-dq.github.io/aegis-dq](https://aegis-dq.github.io/aegis-dq)**.

## Getting help

| Channel | Use for |
|---|---|
| [GitHub Issues](https://github.com/aegis-dq/aegis-dq/issues) | Bug reports, feature requests |
| [GitHub Discussions](https://github.com/aegis-dq/aegis-dq/discussions) | Questions, ideas, show-and-tell |
| [Stack Overflow `aegis-dq`](https://stackoverflow.com/questions/tagged/aegis-dq) | How-to questions |

## Before opening an issue

1. Search [existing issues](https://github.com/aegis-dq/aegis-dq/issues) — your question may already be answered
2. Check the [documentation](https://aegis-dq.github.io/aegis-dq)
3. Include your `aegis-dq` version (`pip show aegis-dq`), Python version, and a minimal reproducible example

## GitHub Action support

For issues specific to the `aegis-dq/aegis-dq` GitHub Action, include:

- The full workflow YAML (redact secrets)
- The step output / error message
- The `aegis-dq` version pinned in your workflow
